# CS-370_Current_and_Emerging_Trends
Deep Q-Learning (DQN) implementation of a pirate intelligent agent that learns to navigate a maze environment using reinforcement learning and neural networks.

This project implements a Deep Q Learning DQN pirate intelligent agent that learns to navigate a maze environment through reinforcement learning and neural networks. I built and refined the training loop, implemented epsilon greedy exploration with decay, integrated experience replay and a target network for stability, and tuned hyperparameters to improve convergence and performance tracking. Through this work, I applied core computer science principles such as algorithm design, systematic debugging, performance evaluation, and iterative optimization to develop an adaptive decision making system. This project reflects how computer scientists approach complex problems by breaking them into structured components, testing methodically, and improving systems based on measurable results. It also reinforces the ethical responsibility to design AI systems that are safe, transparent, and reliable, since reinforcement learning agents make decisions based on reward structures that can significantly influence behavior.
